How to use this calculator

Use the By Area tab if you know the finished size of your beaded design. Use the By Grid tab if you have a grid pattern and know the exact bead count per row and column. Results include a 15% waste buffer for rejects, dropped beads, and color sorting.

  • Size 11/0 seed beads are the most common for bead embroidery on fabric
  • Coverage % lets you account for partially beaded areas — e.g. 60% if half the design is bare fabric
  • For multi-color designs, divide total bead count by the number of colors as a starting estimate

Bead counts are estimates. Actual usage varies by technique (brick stitch, peyote, backstitch) and bead lot size variation.
