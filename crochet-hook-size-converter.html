Understanding Hook Size Systems

  • US: Uses a letter/number combo (B/1, C/2…). Larger letters = larger hooks.
  • Metric (mm): The most precise system — measures the actual shaft diameter. Used internationally.
  • UK/Canadian: An older numbering system where higher numbers mean smaller hooks — the opposite of the US system. Less common today.
  • When patterns list only one system, use the mm measurement to match accurately.
