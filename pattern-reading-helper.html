Pattern Reading Helper

Paste a pattern row or round and see abbreviations expanded to plain English.

Plain-English Expansion

This tool uses rule-based pattern recognition. Always verify with the original pattern.

How to Use

  1. Select your craft type (Crochet or Knitting).
  2. Paste or type a single row or round instruction.
  3. Click Expand Pattern to see abbreviations expanded inline.

Tips

  • Separate instructions with commas for best results.
  • Abbreviations are case-insensitive (DC, dc, Dc all work).
  • Unknown abbreviations will be flagged so you can look them up.
