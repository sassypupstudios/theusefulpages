How to use

  • Tap the large + button to count up one row or round.
  • Use the − button to undo the last count.
  • Give your counter a name (e.g., "Left Sleeve" or "Body").
  • Set a target row count to see a progress bar.
  • Up to 5 counters can be saved at once — great for tracking multiple pieces.
  • All counters are saved automatically to your browser — they'll still be here when you return.
  • Use Reset to set a counter back to 0, or Delete to remove it.
