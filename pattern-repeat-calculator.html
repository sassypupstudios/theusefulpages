Pattern Repeat Calculator

Check if your stitch count fits a pattern repeat, find adjusted counts, and get a written cast-on formula with optional edge stitches.

Edge stitches are worked outside the pattern repeat on each side.

How to use

  • Enter your total stitch count — the number currently on your needles or that you plan to cast on.
  • Enter your pattern repeat size — for example, if the pattern says "8-stitch repeat," enter 8.
  • Optionally enter edge stitches per side — these are plain stitches that frame the pattern (e.g., selvedge stitches).
  • The result shows how many full repeats fit, how many leftover stitches there are, and the nearest valid counts.
  • The written formula shows how the stitches break down as a cast-on instruction.

Example: For a 100-stitch count with an 8-stitch repeat and 2 edge stitches each side:
Available for repeats = 100 − 4 = 96 stitches. 96 ÷ 8 = 12 full repeats, 0 leftover. Formula: "2 edge + (8 × 12) + 2 edge = 100 stitches."
