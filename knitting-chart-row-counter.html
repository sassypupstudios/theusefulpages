Knitting Chart Row Counter

Track multiple charts simultaneously. Progress saves automatically to your browser.

Add a New Chart

How to Use

  1. Enter a chart name, total rows, and repeat length, then click + Add Chart.
  2. Use the + button to advance one row, − to go back.
  3. The progress bar, current row, and repeat position update automatically.
  4. Add notes to each chart using the notes field.
  5. All progress saves automatically to your browser (localStorage) — close and reopen safely.

Tips

  • Add multiple charts if your project has several different chart sections (e.g., body, sleeves, border).
  • Use the Export button to copy your progress as plain text for safekeeping.
  • The repeat position tells you which row within the current repeat you are on.
