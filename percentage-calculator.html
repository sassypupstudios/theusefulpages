How to use

  • Select a calculation mode using the tabs above.
  • X% of Y — Enter a percentage and a number to find what that percentage equals.
  • % Change — Enter a starting value and ending value to find the percentage increase or decrease.
  • What % is X of Y — Enter two numbers to find what percentage the first is of the second.
  • Reverse % — Enter a value and a percentage to find what the original whole number was.
  • Results update on calculate. Use the Copy Result button to copy the answer to your clipboard.
