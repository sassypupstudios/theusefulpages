How to use

  • Type or paste a repeat instruction from your pattern.
  • Click an example chip to prefill with a common pattern format.
  • Click Expand Repeat to see every stitch written out.
  • The tool shows the first 3 iterations in full, then notes how many more remain.
  • Total stitch count counts each stitch token in the full expansion.
  • Switch between Crochet and Knitting mode for mode-specific examples.
  • Chains (ch) count as stitches for the purposes of the total count shown.
Supported formats: *sc, dc* repeat 4 times — [k2, p2] 6 times — (dc, ch1, dc) repeat to end — repeat from * to end 5 times
