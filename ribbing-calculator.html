How to use

  • Enter the desired width and your gauge from a stockinette/main stitch swatch.
  • Select your ribbing pattern — the calculator adjusts the stitch count to fit the repeat.
  • The result shows a written row instruction for both RS and WS rows.
  • Ribbing is usually worked on needles one size smaller than the main fabric for a neater, tighter rib.

Ribbing pulls in narrower than stockinette. The actual finished width of a ribbed section can be 10–20% narrower than the calculated width when relaxed.
