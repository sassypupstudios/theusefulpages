What Is Fabric Count?

Fabric count refers to the number of stitches (or threads) per inch. A 14-count Aida has 14 holes per inch, so a 140-stitch-wide design will measure exactly 10 inches across.

28-count and 32-count "Stitched Over 2"

Evenweave fabrics like 28-count linen are stitched over 2 fabric threads, which makes each cross-stitch the same size as on 14-count Aida. Similarly, 32-count stitched over 2 equals 16-count Aida.

Formula

Finished size (inches) = Stitch count ÷ Fabric count. Always add at least 3 inches (7.5 cm) on each side of the design area when buying fabric.
