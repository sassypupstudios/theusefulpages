How to use

  • Enter an event name and select a target date, or use a quick preset button.
  • Click Start Countdown to see a live countdown in days, hours, minutes, and seconds.
  • If the date is in the past, it shows how many days ago it was.
  • Click Save Event to store it in your browser — saved events persist across visits.
  • Click any saved event to reload it as the active countdown.
  • For birthdays, click the Birthday preset, enter the month and day, then click Set Birthday.
