Pattern Error Checker

Paste your crochet or knitting pattern and scan for common issues — unmatched brackets, unknown abbreviations, stitch count mismatches, and more.

What this tool checks

  • Unmatched brackets — checks that every (, [, and { has a matching closing bracket.
  • Unmatched asterisks — flags a lone * that doesn't have a paired closing *.
  • Unknown abbreviations — any ALL-CAPS word not in the built-in dictionary of 60+ common crochet and knitting abbreviations is flagged.
  • Stitch count mismatch — if a line states a count like (18 sts), the tool attempts to count stitches in that line and flags large discrepancies.
  • Missing turn/chain — warns if rows appear to start without a turning chain or turn instruction.
Disclaimer: This is a rule-based checker, not a professional pattern editor. It cannot understand pattern intent, and results should be verified by the crafter. False positives are possible — especially with custom abbreviations or unusual formatting.
